# CodeAnt AI Frontend Intern Assignment

## How to Run It

### Clone the Repository

In your terminal:

```bash
git clone https://github.com/your-username/your-repository.git
cd your-repository

npm install

npm start
```

Open your browser and navigate to:
http://localhost:3000
